# Setup

本目录包含与 RenkoPlayer 安装、先决条件及发布/部署相关的说明文档。

包含文件：

- `prerequisites.md` — 开发与运行所需的先决条件（系统、工具、依赖）。
- `install_windows.md` — 在 Windows 上的详细安装与本地运行步骤（含 vcpkg、CMake、构建说明）。
- `deploy_release.md` — 发布构建与部署建议（打包 qrc、安装目录、可执行分发）。

建议：阅读 `prerequisites.md` 以确认环境，然后按照 `install_windows.md` 进行开发构建；完成后参考 `deploy_release.md` 制作可分发产物。
